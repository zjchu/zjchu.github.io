CorrTalk: Correlation Learning in Diverse Facial Activity and Hierarchical Speech for 3D Animation
Zhaojie Chu, Kailing Guo, Xiaofen Xing, Yilin Lan, Bolun Cai, Xiangmin Xu

arXiv  /  Project  /  Code  /  Online demo

We minimize the over-smoothed facial expression using a learned discrete motion prior, which means more dramatic and expressive facial motions with more accurate lip movements can be achieved.
